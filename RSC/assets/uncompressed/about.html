Rust Server Creator

by x04000

Rust Server Creator (RSC) is an Open Source project to make server more easier

The code are witted in C# and the server base is SteamCMD (from Valve)

RSC Support plugins with Oxide Mod (from uMod)

Most regular issues

If the UI and text of the program sometimes crash or glitch, try to install the text fonts (they are in RSC\[Rust/Valorant]_Font.ttf)

If you have problems during the installation of SteamCMD or Oxide Mods, maybe you need to disable the antivirus (after the installation activate it again)

RSC it's sometimes detected by the antivirus, this is because RSC it's a fan project
