Kondor2 Commands

menu -> Displays Kondor Menu | help / commands -> Shows this list | exit -> Exit the application | clear / clean / cls -> Clean the Kondor console | general -> View the general menu | language -> Change the language | ip / IP -> IP related options | prip -> Show your private IP | pbip -> Show your public IP | mip -> Show your IP | dmip -> Show detailed info of your IP | logout -> Logout of Kondor | security -> Displays the security menu | sfc -> Scan the SFC (System File Checker) | dism -> Scan the DSIM (Deployment Image Servicing and Management) | rdism -> Restore the health of the DISM | acpr -> Show the list of the active process | fspr -> Force the stop of a process | prex -> Advanced process administrator (ProcessExplorer) | fw -> Enable/Disable firewall of Windows | tools -> Displays the tools menu | secrets -> Secret folder | noadmin -> Execute program as admin without their permission | addons -> Displays the addons menu | addonslist -> View the installed addons list | portablepython -> Python portable version | attacks / attack -> Display the attacks menu | ddos -> DDoS Panel
