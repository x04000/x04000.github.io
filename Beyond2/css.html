CSS3

Con las hojas de estilos CSS podemos describir como se van a ver los elementos de la web, como poner botones de X manera, crear animaciones, tener un estilo común en los textos etc.

Es básicamente una forma de cambiar la presentación de la web y tener un estilo personalizado y tener varios elementos con el mismo estilo en común etc.
